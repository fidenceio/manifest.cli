# Release v39.0.0

**Release Date:** 2026-04-04 19:20:20 UTC
**Release Type:** Major

---

## What's New in v39.0.0

This major release introduces **smart `.gitignore` management** and **fleet auto-discovery by default**,
making project initialization and polyrepo workflows significantly smoother.

### Smart `.gitignore` Scaffolding

Manifest now automatically ensures every project has a well-configured `.gitignore`.
The new `ensure_gitignore_smart()` function handles three scenarios:

| Scenario | Action |
| -------- | ------ |
| No `.gitignore` exists | Creates one with best-practice defaults |
| `.gitignore` exists but is empty | Overwrites with defaults (with deferred warning) |
| `.gitignore` has entries | Creates `.gitignore.manifest` as a merge reference |

The default template covers OS files, editor configs, environment secrets, dependency caches,
build outputs, test coverage, compiled files, Terraform state, Docker overrides, and more.

### Fleet Auto-Discovery by Default

`manifest fleet init` now **discovers Git repositories automatically** when initializing a fleet.
Previously, discovery required an explicit `--discover` flag. The new behavior:

- **Default**: scans your workspace for Git repos and adds them to `manifest.fleet.yaml`
- **`--bare`**: skip discovery entirely and create a minimal fleet template
- Each discovered repo gets a `.gitignore` check (created or `.gitignore.manifest` reference)
- Summary with per-repo status (created, preserved, failed) displayed after initialization

### Deferred Warning Pattern

Warnings for edge cases (like overwriting an empty `.gitignore`) are now collected and displayed
**after** the operation summary, rather than interrupting the workflow inline.

---

## Breaking Changes

- `manifest fleet init` runs auto-discovery by default (was opt-in via `--discover`)
- The `--discover` and `--no-discover` flags have been replaced by `--bare`

## Installation

```bash
# Homebrew (recommended)
brew tap fidenceio/tap
brew install manifest

# Or upgrade an existing installation
brew update && brew upgrade manifest

# Or via install script
curl -fsSL https://raw.githubusercontent.com/fidenceio/manifest.cli/main/install-cli.sh | bash
```
