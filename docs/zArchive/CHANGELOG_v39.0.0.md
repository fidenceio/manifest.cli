# Changelog v39.0.0

**Release Date:** 2026-04-04 19:20:20 UTC
**Release Type:** Major

---

## New Features

- **Smart `.gitignore` scaffolding** (`ensure_gitignore_smart()`)
  - Three-case detection: missing file, empty file, file with entries
  - Creates `.gitignore.manifest` reference when existing ignore rules are present
  - Output signaling via stdout: `.gitignore`, `.gitignore:empty-overwrite`, `.gitignore.manifest`
  - Logging functions write to stderr, so command substitution captures only signals

- **Best-practice `.gitignore` template** (`create_default_gitignore()`)
  - Sections: Manifest CLI, OS files, Editor/IDE, Environment/secrets, Logs, Dependencies,
    Package manager caches (Yarn PnP-safe), Build outputs, Test/coverage, Compiled files,
    Temporary files, Archives, Terraform, Docker, Miscellaneous
  - Includes `!.env.example` and `!.env.template` exceptions
  - Self-cleaning: `.gitignore.manifest` is in the template itself

- **Fleet auto-discovery by default**
  - `manifest fleet init` now runs `discover_fleet_repos()` automatically
  - `--bare` flag skips discovery (replaces the old `--discover` / `--no-discover`)
  - Per-repo `.gitignore` check during fleet initialization
  - Deferred warnings for empty-overwrite repos listed after summary

## Improvements

- `ensure_required_files()` uses `ensure_gitignore_smart()` with deferred warning pattern
- Fleet init gitignore loop tracks created/overwritten/reference/skipped/failed counts
- Overwritten repo names collected and displayed in a deferred warning block
- Error handling on write failures with `return 1` propagation

## Breaking Changes

- `manifest fleet init` default changed from no-discovery to auto-discovery
- `--discover` / `--no-discover` flags removed in favor of `--bare`

## Files Changed

- `modules/core/manifest-shared-functions.sh` — added `ensure_gitignore_smart()`,
  rewrote `create_default_gitignore()`, updated `ensure_required_files()`, updated exports
- `modules/fleet/manifest-fleet.sh` — rewrote `fleet_init()` with auto-discovery default,
  `--bare` flag, gitignore loop with deferred warnings
